# Totem Purchase

## 📱 Sobre o Projeto

**Totem Purchase** é um sistema de vendas integrado a um totem de auto-atendimento para empresas. A aplicação foi desenvolvida para facilitar e automatizar o processo de vendas através de uma interface interativa e intuitiva em um totem digital.

## 🎯 Objetivo

O sistema permite que os clientes realizem compras de forma autônoma através de um totem de auto-atendimento, integrando todo o fluxo de vendas, gestão de produtos, carrinho de compras e pagamento em uma única plataforma moderna e eficiente.

## ✨ Funcionalidades

- 🛍️ Catálogo de produtos integrado
- 🛒 Carrinho de compras intuitivo
- 💳 Integração com sistemas de pagamento
- 📊 Gestão de vendas em tempo real
- 🔐 Autenticação segura
- 📈 Relatórios e análises de vendas
- 🖥️ Interface responsiva para totem
